# a mountain out of a molehill

play it [here](https://nervous-koala.itch.io/a-mountain-out-of-a-molehill)!
this game was made for the [godot wild jam #90](https://itch.io/jam/godot-wild-jam-90)

```
Climb the mountain (counter-clockwise) to reach the top

WASD to move

mouse to aim stick

Click to attack

hold click for heavy attack

space to dash
```
